# Simple Calculator VB

Simple Calculator VB adalah aplikasi kalkulator aritmatika berbasis desktop yang dikembangkan menggunakan Visual Basic .NET. Proyek ini dirancang dengan antarmuka grafis (GUI) yang intuitif untuk memberikan pengalaman pengguna yang sederhana namun efektif dalam melakukan perhitungan matematika dasar.

## 🌟 Fitur Utama

- **Operasi Aritmatika Dasar:** Mendukung penjumlahan (+), pengurangan (-), perkalian (x), dan pembagian (/).
- **Real-Time Processing:** Menerapkan logika matematika yang presisi untuk pengolahan angka secara langsung.
- **User-Friendly Interface:** Menggunakan kontrol `Button` dan `TextBox` yang tersusun rapi untuk kemudahan navigasi.
- **Error Handling Sederhana:** Dirancang untuk menangani input angka dasar secara efisien.

## 🛠️ Teknologi & Peralatan

- **Bahasa Pemrograman:** Visual Basic .NET (VB.NET)
- **Framework:** .NET Framework
- **IDE:** Visual Studio

## 📋 Prasyarat

Untuk menjalankan atau memodifikasi proyek ini, Anda memerlukan:
- Microsoft Visual Studio (versi 2012 atau lebih baru).
- .NET Framework yang terinstal di sistem Windows Anda.
